```
copy /b test.png + src.zip target.png
```
```
将src.zip 和test.png合并乘新的图片target.png,target.png保留test.png的像素信息,用图片查看器看的话，跟原来的一样。  
同时如果将target.png后缀名改成.zip解压,会把src.zip内的文件解压出来
```

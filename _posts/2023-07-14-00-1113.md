```
1、os.clock 返回负值
在某些32位的设备上,os.clock会在一定的时间后返回负值,从而导致两个变量间计算差值出现错误
```
```
2、string.find 要查找的参数如果带有模式匹配相关的符号要特殊处理
local idx = string.find("a-b+c","b+c")
print("idx====",idx)
output: idx==== nil

修改:
local idx = string.find("a-b+c","b+c",nil,true)
print("idx====",idx)
output: idx==== nil

第4个参数为true 关闭模式匹配,使用简单字符串匹配
```
